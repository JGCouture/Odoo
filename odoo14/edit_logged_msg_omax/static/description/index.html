Edit/Delete Messages and Notes

Overview

This App allow user to edit and delete its own messages/notes and other users messages/notes.

Features

  • Option to Edit and Delete messages/notes
  • Easy to Edit with Icon
  • Easy to Delete with Icon
  • Aceesrights for user to edit/delete own messages/notes
  • Easy to Edit with 'Pencil' Icon
  • Aceesrights to Allow user to delete and edit messages/notes of other users
  • Maintain history for edited messages
  • Edited message will highlighted with Icon
  • Editing/Deleting would be available for all communication means: documents threads, mail channels, direct discussion, live chats.
  • Only user who has accessrights can perform editing/deleting operations.
  • Only user who has accessrights can perform editing/deleting operations.
  • We believe subject is not suppose to change so we do not provide editing subject of message/notes
  • System generated messages will not have an oprtion to edit/delete
